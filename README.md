# md

**Читалка Markdown прямо в терминале.** Один файл, чистый Python, ноль зависимостей.
Красивый рендер, мгновенное листание одной клавишей, переходы по ссылкам —
включая вики-ссылки `[[вот такие]]`.

Удобно и разработчику (vim-клавиши, подсветка синтаксиса, поиск), и бабушке
(стрелки, крупные подсказки на экране, понятный русский интерфейс).

```
python md.py README.md
python md.py ./заметки/       # браузер по папке
python md.py                  # текущая папка
```

---

## Что умеет

| | |
|---|---|
| 🎨 **Рендер** | Заголовки, списки, чек-боксы, цитаты, таблицы, горизонтальные линии, YAML front-matter |
| ✨ **Инлайн** | **Жирный**, *курсив*, ~~зачёркнутый~~, `код`, ==выделение== |
| 🧑‍💻 **Код** | Подсветка синтаксиса (Python, Bash, JS/TS, JSON, …) с номерами строк в рамке |
| 🔗 **Ссылки** | `[[вики-ссылки]]`, `[[ссылка\|с алиасом]]`, обычные `[текст](url)` — с нумерацией и переходом |
| 📖 **Листание** | Мгновенная навигация одной клавишей: стрелки, `PgUp`/`PgDn`, `g`/`G`, vim `j`/`k` |
| 🔎 **Поиск** | Поиск по тексту с переходом между совпадениями (`/`, `n`, `N`) |
| 📁 **Браузер** | Навигация по папкам с заметками стрелками |
| ✏️ **Правка** | Открыть файл в `$EDITOR` (или `nano`/`vi`/`notepad`) прямо из просмотра |
| 🌗 **Совместимость** | Автоопределение Unicode / OSC 8-гиперссылок; аккуратный ASCII-фолбэк для старых консолей |

Никаких `pip install`. Нужен только Python 3.8+.

## Запуск

```bash
python md.py <файл.md>     # открыть конкретный файл
python md.py <папка>       # открыть браузер по папке (vault)
python md.py               # то же для текущей папки
python md.py --help        # справка
```

Когда вы указываете папку, она становится «хранилищем» (vault): вики-ссылки
`[[Имя заметки]]` ищутся по всем `.md`-файлам внутри неё рекурсивно.

## Управление

### Просмотр файла

| Клавиши | Действие |
|---|---|
| `↓` `j` `Enter` | строка вниз |
| `↑` `k` | строка вверх |
| `Пробел` `PgDn` `f` | страница вниз |
| `b` `PgUp` | страница вверх |
| `g` `Home` | в начало |
| `G` `End` | в конец |
| `0`–`9` | перейти по ссылке с этим номером |
| `/` | поиск по тексту |
| `n` / `N` | следующее / предыдущее совпадение |
| `e` | редактировать файл |
| `r` | обновить |
| `?` `h` | справка |
| `q` `Esc` | назад / выход |

### Браузер по папке

| Клавиши | Действие |
|---|---|
| `↑` `↓` | выбор пункта |
| `Enter` `→` | открыть файл или папку |
| `←` `Backspace` | на уровень вверх |
| `q` `Esc` | выход |

> Раскладка не мешает: работают и русские клавиши на тех же кнопках
> (`й` = `q`, `у` = `e`, и т.д.).

## Ссылки

Каждая ссылка в тексте помечается номером в квадратных скобках. Чтобы перейти —
просто нажмите её номер.

- **Вики-ссылки** `[[Другая заметка]]` открывают соответствующий `.md`-файл.
  Если заметки ещё нет — предложат создать её.
- **Алиасы** `[[Файл|как показать]]` — отображается алиас, переход по файлу.
- **Обычные ссылки** `[текст](https://…)` открываются в браузере. В современных
  терминалах они ещё и кликабельны напрямую (OSC 8).

## English (TL;DR)

A zero-dependency, single-file terminal Markdown reader written in pure Python.
Rich rendering, instant single-key paging, syntax-highlighted code blocks, full
text search, and link navigation — including Obsidian-style `[[wikilinks]]`.
Works both for developers (vim keys, search) and for absolute beginners (arrow
keys, on-screen hints).

```bash
python md.py <file.md>   # open a file
python md.py <folder>    # browse a vault of notes
```
